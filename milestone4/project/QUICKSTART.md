# Quick Start Guide

## 1. Install Dependencies
```bash
pip install -r requirements.txt
```

## 2. Run the Application
```bash
python app.py
```

## 3. Open in Browser
Navigate to: `http://localhost:5000`

## 4. Start Using
- Show your hand to the webcam
- Bring thumb and index finger together to adjust volume
- Use calibration controls if needed
- Monitor metrics in real-time

## System Requirements
- Python 3.8+
- Webcam
- Windows/Mac/Linux

## Key Features
✓ Real-time hand tracking (256×192, 8-10 FPS)
✓ 21 hand landmarks with visual overlay
✓ Gesture recognition (Pinch, Open Hand, Closed Fist)
✓ Live distance graph
✓ Auto/Manual calibration
✓ Performance metrics dashboard
✓ Professional responsive UI
